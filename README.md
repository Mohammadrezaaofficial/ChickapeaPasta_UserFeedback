# ChickapeaPasta UserFeedback Data Science Project

A Company name Chickapea came to us with an objective of driving trial and brand-awareness among health-oriented consumers in urban markets. Peekage designed a study and sent a free sample of product to the users.

After running this campaign for the company, we gathered some data from the users regarding the product. You can see the context of the campaign we ran and the product that was tested by users below.

My mission is to report to company about the result of the campaign, the questions and how did the product perform according to the campaign's objective.

This campaign promotes **Chickapea Spirals** pasta across 27 U.S. states. The product—made from chickapeas and yellow peas—delivers **24 g protein** and **8 g fiber** per 100 g serving, offering a nutritious yet familiar alternative to wheat pasta.

I will:
- (A) **clean** the dataset with transparent, reproducible rules. 
- (B) produce a concise **brand report** answering the brief.
- (C) build a **reporting agent** that answers marketer questions **only from the cleaned data**.
- (D) Report the answering datas as json file
- (E) Create a Mysql dataset on answering datas
